# Hotwire

Turbo Drive, Frames, Streams, Stimulus, and Strada patterns for Rails. Covers single-page-app-like navigation, partial page updates, real-time broadcasting, and native bridge integration — all without writing custom JavaScript.

## Skills

- `turbo_drive.md` — Full-page navigation with Turbo Drive
- `turbo_frames.md` — Scoped navigation and lazy loading with Turbo Frames
- `turbo_streams.md` — Partial page updates over HTTP and WebSockets
- `stimulus_controllers.md` — Writing and organizing Stimulus controllers
- `stimulus_values_targets.md` — Declaring values, targets, and classes in Stimulus
- `morphing.md` — DOM morphing with Turbo 8 page refreshes
- `broadcasting.md` — Real-time Turbo Stream broadcasts via Action Cable
- `lazy_loading_frames.md` — Deferred loading with Turbo Frame lazy src
- `form_submissions.md` — Handling form submissions with Turbo
- `page_refreshes.md` — Turbo 8 morphing-based page refreshes
- `custom_actions.md` — Defining custom Turbo Stream actions
- `testing_turbo.md` — System and integration tests for Turbo-driven UIs
- `stimulus_outlets.md` — Cross-controller communication with Stimulus outlets
- `turbo_native_strada.md` — Turbo Native and Strada bridge components

## Installation

```
/install-skills hotwire
```

## Compatibility

- Rubyn Code >= 0.5.0
- Rails >= 7.0
- Gems: `turbo-rails`, `stimulus-rails`
