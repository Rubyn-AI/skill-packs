# Kamal

Container deployment, accessories, SSL proxy, secrets management, multi-server setups, asset bridging, and monitoring for Rails applications deployed with Kamal.

## Skills

- `deployment_basics.md` — Initial setup and deploying with Kamal
- `accessories.md` — Running databases, Redis, and services as accessories
- `ssl_proxy.md` — Automatic SSL with Kamal's built-in proxy
- `secrets.md` — Managing environment variables and secrets
- `multi_server.md` — Multi-role and multi-server deployments
- `asset_bridging.md` — Zero-downtime asset serving across deploys
- `monitoring.md` — Health checks, logging, and deployment monitoring

## Installation

```
/install-skills kamal
```

## Compatibility

- Rubyn Code >= 0.5.0
- Rails >= 7.0
- Gems: `kamal`
