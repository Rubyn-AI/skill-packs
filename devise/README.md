# Devise

Authentication setup, OmniAuth social login, JWT API auth, Turbo compatibility, and security hardening for Rails applications using Devise.

## Skills

- `basic_setup.md` — Installing and configuring Devise
- `omniauth.md` — Social login with OmniAuth providers
- `jwt_api_auth.md` — Token-based API authentication with JWT
- `confirmable_lockable.md` — Email confirmation and account lockout
- `custom_strategies.md` — Writing custom Warden authentication strategies
- `turbo_compatibility.md` — Making Devise work with Turbo Drive
- `testing_devise.md` — Testing authentication flows
- `security_hardening.md` — Password policies, rate limiting, and secure defaults

## Installation

```
/install-skills devise
```

## Compatibility

- Rubyn Code >= 0.5.0
- Rails >= 7.0
- Gems: `devise`
