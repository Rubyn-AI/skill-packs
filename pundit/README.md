# Pundit

Policy classes, scopes, testing, headless policies, namespacing, and controller integration patterns for authorization in Rails.

## Skills

- `policy_basics.md` — Writing and applying Pundit policy classes
- `scopes.md` — Policy scopes for collection filtering
- `testing_policies.md` — RSpec and Minitest patterns for policy specs
- `headless_policies.md` — Policies without an associated record
- `namespaced_policies.md` — Policies for namespaced and STI models
- `integration_patterns.md` — Controller integration and `verify_authorized`

## Installation

```
/install-skills pundit
```

## Compatibility

- Rubyn Code >= 0.5.0
- Rails >= 7.0
- Gems: `pundit`
