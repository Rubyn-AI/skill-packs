# GraphQL-Ruby

Schema design, resolvers, mutations, subscriptions, DataLoader, authorization, and performance patterns for the graphql-ruby gem in Rails.

## Skills

- `schema_design.md` — Structuring types, queries, and schema organization
- `resolvers.md` — Writing resolver classes and field resolution
- `mutations.md` — Input types, mutation classes, and error handling
- `subscriptions.md` — Real-time subscriptions with Action Cable
- `authorization.md` — Field-level and type-level authorization
- `dataloader.md` — Batch loading with GraphQL::Dataloader
- `pagination.md` — Cursor-based and offset pagination with connections
- `testing_graphql.md` — Testing queries, mutations, and subscriptions
- `performance.md` — Query complexity limits, caching, and N+1 prevention

## Installation

```
/install-skills graphql-ruby
```

## Compatibility

- Rubyn Code >= 0.5.0
- Rails >= 7.0
- Gems: `graphql`
