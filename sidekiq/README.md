# Sidekiq

Job patterns, queue management, retry strategies, batches, scheduling, testing, and performance tuning for Sidekiq background processing in Rails.

## Skills

- `job_patterns.md` — Idempotent, small, and composable job design
- `queues_priorities.md` — Queue configuration and priority management
- `retry_strategies.md` — Custom retry logic and dead-letter handling
- `batches.md` — Sidekiq Pro/Enterprise batch workflows
- `scheduled_jobs.md` — Recurring and delayed job scheduling
- `testing_sidekiq.md` — Testing jobs with inline and fake modes
- `monitoring.md` — Sidekiq Web UI, metrics, and alerting
- `performance.md` — Concurrency tuning, connection pooling, and throughput

## Installation

```
/install-skills sidekiq
```

## Compatibility

- Rubyn Code >= 0.5.0
- Rails >= 7.0
- Gems: `sidekiq`
