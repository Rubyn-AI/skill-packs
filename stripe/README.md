# Stripe

Payment processing, subscriptions, webhooks, Checkout Sessions, Stripe Connect, and Pay gem integration for Rails applications.

## Skills

- `webhooks.md` — Receiving and verifying Stripe webhook events
- `checkout_sessions.md` — Hosted and embedded Checkout flows
- `subscriptions.md` — Recurring billing and plan management
- `customer_portal.md` — Self-service billing portal integration
- `idempotency.md` — Idempotency keys for safe retries
- `testing.md` — Testing with Stripe fixtures and mocks
- `pay_gem_integration.md` — Using the Pay gem for multi-provider billing
- `error_handling.md` — Handling Stripe API errors gracefully
- `strong_customer_authentication.md` — SCA and 3D Secure compliance
- `connect_platforms.md` — Stripe Connect for marketplace platforms
- `metered_billing.md` — Usage-based billing and metered subscriptions

## Installation

```
/install-skills stripe
```

## Compatibility

- Rubyn Code >= 0.5.0
- Rails >= 7.0
- Gems: `stripe`
