# ViewComponent

Component structure, slots, previews, testing, Phlex comparison, and Stimulus integration for building reusable UI components in Rails.

## Skills

- `component_basics.md` — Creating and rendering ViewComponent classes
- `previews.md` — Component previews and Lookbook integration
- `testing_components.md` — Unit and integration testing for components
- `slots.md` — Content slots for flexible component composition
- `phlex_comparison.md` — ViewComponent vs Phlex trade-offs and migration
- `stimulus_integration.md` — Wiring Stimulus controllers into components
- `patterns.md` — Common component patterns and composition strategies

## Installation

```
/install-skills view-component
```

## Compatibility

- Rubyn Code >= 0.5.0
- Rails >= 7.0
- Gems: `view_component`
